# Sujal Shah MERN Portfolio

A modern, responsive portfolio website built with the MERN stack (MongoDB, Express, React, Node.js).

## 🌐 Live Website

**[View Live Portfolio](https://sujal-shah-portfolio-1020549552082.asia-southeast1.run.app/)**

https://sujal-shah-portfolio-1020549552082.asia-southeast1.run.app/

## 🚀 Features

- Modern, responsive UI design
- Smooth animations and transitions
- Project showcase with case studies
- Skills section
- About section
- Contact section
- Mobile-friendly navigation

## 📦 Tech Stack

- **Frontend**: React + TypeScript + Tailwind CSS
- **Build Tool**: Vite
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/Sujalshah20/Sujal-portfolio.git
cd Sujal-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser
